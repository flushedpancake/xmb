What's New No changelog information exists for 1.9.8 Engage. 1.9.8 Engage's codebase was rolled back and is based on John Brigg's 1.9.5 SP1. This new codebase incorporates a multitude of bug and security fixes and is considered to be more secure than 1.9.7 Nexus.

  • MySQL 5 compatibility
