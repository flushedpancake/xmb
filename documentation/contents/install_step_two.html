Installation Step Two :: Upload Files First, create a directory for your forum in your web space.

All files and sub-directories in the "Files" directory of the download package require uploading to the newly created directory in your web space.

Binary mode should be used for transferring the files. Most FTP clients will do this automatically, but if you are using a command line FTP client, use the “bin” command prior to transferring.
