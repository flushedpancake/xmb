Since XMB 1.8 SP3 has been released, XMB 1.9.1 has had more than 100 new features added. Here's a small sample.

New User Features

  • New U2U Interface
    Still like the old U2U interface? Change in through Control Panel/Options->Edit Profile
  • New Address Book
    For keeping your contacts on your forums.
  • Spell Checker for posts (requires PHP to be integrated with aspell - ask your hoster!)
  • New themes
  • Quick Reply With random smilie Insertion
  • Pause News Ticker
  • Webcam Option (Url Input In Profile)
  • Spaced Categories
  • Fully XHTML 1.0 Valid Templates

New Administration Features

  • PHP/MySQL Usage Output in footer
  • New Admin Panel interface (with special thanks to John Briggs). Some of the new tools:
    • Prune Thread
    • Prune actually works (and quickly too!)
    • Empty Thread
    • Copy Thread
    • Rename users
    • Analyze database
    • Optimize database
    • Fix Last Post Tool
    • Find and move orphaned threads
    • Find and delete orphaned attachments
    • Backup without using any other tools (need a fast Internet connection for larger forums)
    • Avatar Size Control
    • Rank Editing "on/off" Control (Contains users to edit only their posts; for staff they can only edit from their rank down.)
    • gzip compression on by default (saves more bandwidth)
    • Comment Output "on/off" (off by default - saves bandwidth)
    • Staff Avatar Option

New hoster-friendly features

  • Professional quality security review completed - vastly more secure than XMB 1.8 SP3
  • Saves about 40-50% bandwidth over XMB 1.8 SP3
  • Can handle about twice the number of simultaneous users on the same host as XMB 1.8 SP3
  • All SQL queries are fully specified, use compact indexes, and return minimal data. Reduces MySQL load by sometimes up to 540x over 1.8 SP3
  • PHP5 Compliant
